The dataset dir for finetune.
Each speaker's data is placed under it's own folder.

All wav files are 44.1khz audio.

The .lab annotation file contains the transcription of the audio, with no special formatting.

```
.
├── Speaker-1
│   ├── 1.lab
│   ├── 1.wav
│   ├── 2.lab
│   ├── 2.wav
│   ├── 3.lab
│   └── 3.wav
└── Speaker-2
    ├── 1.lab
    └── 1.wav
```
